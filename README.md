# Odin Rock-Paper-Scissors

## The project
The project consists of a single page, on which you can play RPS with a computer. The game plays as a best-of-5 - the first one to win 3 rounds wins.

### The Odin Project
The assignment for the project on TOP can be found [here](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors).

Also, the UI was added to the game with another assignment from TOP, which can be found [here](https://www.theodinproject.com/lessons/foundations-revisiting-rock-paper-scissors).
